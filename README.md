# The Writing Palette

A free, self-contained web toolkit for EFL teachers practicing differentiated writing instruction — built around Tomlinson's model of differentiated instruction, with no server, login, or install required.

## What's inside

- **Paragraph Assessment Tool** — an interactive 7-criteria rubric (1–4 scale) that adapts its structural criteria to seven rhetorical modes: Narrative, Descriptive, Expository, Persuasive, Process, Compare & Contrast, and Cause & Effect. Four language criteria (details, grammar, clarity, vocabulary) stay constant across all types. Scores can be displayed out of 5, 10, or 20.
- **Essay Assessment Tool** — the same adaptive architecture, scaled to essay-length structure (introduction, body development, conclusion).
- **Topic Bank** — 1,008 writing prompts spanning all 7 genres × 3 differentiated readiness tiers (Foundational, Developing, Advanced), each tagged with a scaffolding tip, filterable, copyable, and linked directly to its matching rubric.

Both rubric tools generate copy-ready feedback comments for report cards or feedback sheets.

## Using it

Open `index.html` in any browser — no build step, no dependencies beyond two Google Fonts loaded over the network. Works fully offline once the page has loaded once.

## Tech

Plain HTML, CSS, and vanilla JavaScript. No frameworks, no build tools, no external libraries beyond Google Fonts (Fraunces, Karla, Caveat).
